# バグ修正チェックリスト

## 実装前

- [ ] 再現を確認した
- [ ] 再現コマンド、入力、環境、期待値、実際値を記録した
- [ ] Red テストを特定または作成した
- [ ] Red テストが現在のコードで失敗する
- [ ] Red テストの失敗理由が狙い通りである
- [ ] 必要に応じて flaky 挙動を確認した
- [ ] 仮説を書いた
- [ ] 根本原因が証拠で裏付けられている

## 実装中

- [ ] production code の変更は最小限
- [ ] 無関係なリファクタリングなし
- [ ] アサーション削除なし
- [ ] テストの弱体化なし
- [ ] 一時ログは明確に印を付けた
- [ ] 観測コードが business behavior を変えていない

## 実装後

- [ ] Red テストが Green になった
- [ ] 近傍テストが通った
- [ ] 必要に応じて広い検証を実行した
- [ ] 一時ログを削除した、または残す理由を説明できる
- [ ] リファクタリング/後始末は Green 後にだけ行った
- [ ] 後始末後に Red テストを再実行した
- [ ] 最終報告にコマンドと結果を含めた
